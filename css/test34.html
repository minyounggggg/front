        사용법 :
                선택자: 가상클래스{
                    속성: 속성값;
                }

                선택자::가상요소{
                    속성: 속성값;
                }
        
        자주사용하는 가상클래스 ->  :hover, :link, :first-child, :last-child
        자주사용하는 가상요소 ->  ::before, ::after, ::first-line
